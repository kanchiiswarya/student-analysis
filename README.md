# Student Performance Analyzer

A Python project that analyzes student exam scores using Pandas and visualizes insights using Matplotlib.

## Features
- Load student data from CSV
- Calculate average scores per subject
- Identify top scorers
- Detect students scoring below 70
- Visualize results using bar charts

## Tech Stack
- Python
- Pandas
- Matplotlib

## How to Run
```bash
pip install pandas matplotlib
python analysis.py
